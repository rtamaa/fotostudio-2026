# Booking System Laravel

Aplikasi sistem booking dengan fitur manajemen booking, pembayaran, dan admin panel menggunakan Laravel.

---

## 🚀 Fitur Utama

### User
- Registrasi & login
- Melihat kalender booking
- Membuat booking
- Melihat riwayat booking
- Status booking (pending, approved, rejected, completed)

### Admin
- Dashboard admin
- Manajemen booking
- Manajemen user
- Konfirmasi & update status booking
- Generate & create payment record
- Monitoring transaksi pembayaran

### Payment System
- Create payment dari booking
- Tracking status pembayaran
- Integrasi payload transaksi
- Support status: pending, paid, failed

---

## 🛠 Tech Stack

- Laravel 10+
- Livewire (UI interaktif)
- MySQL / MariaDB
- Tailwind CSS (optional)
- Midtrans / payment gateway (optional jika dipakai)
- Blade template
